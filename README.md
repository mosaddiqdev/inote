# iNote

Minimal note-taking app with dark theme, tag organization, and export features.

![iNote Screenshot](./public/showcase.png)

## Features

- Clean, minimal interface
- Dark/Light theme
- Tag-based organization
- Search & filter notes
- Export to PDF or Text
- Auto-save
- Fully responsive

## Tech Stack

- React 19
- React Router
- Vite
- CSS Modules
- LocalStorage

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
